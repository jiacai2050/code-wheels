** Submodule 更新方式

- https://stackoverflow.com/a/914090/2163429

#+begin_src bash

# 修改 .gitmodules 后
git submodule sync

# 更新到最新 commit
git submodule update --init --recursive --remote

# 更新到指定 commit
git submodule update --init
#+end_src
